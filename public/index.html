HTTP Rage

HTTP Rage Status Codes

A memerific HTTP Status Code image service. Not intended to send HTTP methods. This is a simple image placeholder service for your 404, 500, or other page.

Follow @DUZOUR

Available Status Codes

  • 100 - Continue
  • 200 - Ok
  • 302 - Found
  • 401 - Unauthorized
  • 403 - Forbidden
  • 404 - Not Found
  • 405 - Method Not Allowed
  • 406 - Not Acceptable
  • 408 - Request Timeout
  • 409 - Conflict

Usage

Simply include an image on your 404, 500, or other page with a tag like such:

<img src="http://httprage.com/404" />

Or if you need HTTPS:

<img src="https://httprage.com/404" />
